<h1><a name="p-9737944-macos-1" class="anchor" href="https://linux.do#p-9737944-macos-1"></a><img src="https://linux.do/images/emoji/twemoji/hammer_and_wrench.png?v=15" title=":hammer_and_wrench:" class="emoji" alt=":hammer_and_wrench:" loading="lazy" width="20" height="20"> 屏蔽 macOS 系统更新的方法分享</h1>
<p>最近在折腾 macOS 更新机制时，发现了一种<strong>彻底屏蔽系统更新</strong>的办法，实测有效。分享给大家一起交流。</p>
<h2><a name="p-9737944-h-2" class="anchor" href="https://linux.do#p-9737944-h-2"></a><img src="https://linux.do/images/emoji/twemoji/puzzle_piece.png?v=15" title=":puzzle_piece:" class="emoji" alt=":puzzle_piece:" loading="lazy" width="20" height="20">操作步骤</h2>
<ol>
<li><strong>进入系统更新界面</strong><br>
打开「系统设置 → 通用 → 软件更新」，点击“立即更新”。<br>
弹窗出现后，不要点“同意”或“不同意”，<strong>保持停留在此界面</strong>。</li>
<li><strong>修改 Hosts 文件</strong><br>
打开终端并输入以下命令：</li>
</ol>
<pre><code class="lang-auto">sudo nano /etc/hosts
</code></pre>
<p>输入管理员密码后，在文件末尾添加以下内容：</p>
<pre><code class="lang-auto">127.0.0.1 swscan.apple.com
127.0.0.1 swdist.apple.com
127.0.0.1 swdownload.apple.com
127.0.0.1 swcdn.apple.com
127.0.0.1 updates-http.cdn-apple.com
127.0.0.1 updates.cdn-apple.com
127.0.0.1 gdmf.apple.com
127.0.0.1 xp.apple.com
127.0.0.1 mesu.apple.com
</code></pre>
<p>然后按 <strong>Ctrl + O</strong> 保存，<strong>Ctrl + X</strong> 退出编辑器。<br>
3. <strong>返回更新界面</strong><br>
此时点击“同意”，系统会因为无法访问更新服务器而提示“更新失败”，达到屏蔽效果。<br>
4. <strong>刷新 DNS 缓存</strong><br>
执行以下命令清理 DNS 缓存：</p>
<pre><code class="lang-auto">sudo dscacheutil -flushcache
sudo killall -HUP mDNSResponder
</code></pre>
<hr>
<h2><a name="p-9737944-h-3" class="anchor" href="https://linux.do#p-9737944-h-3"></a><img src="https://linux.do/images/emoji/twemoji/red_circle.png?v=15" title=":red_circle:" class="emoji" alt=":red_circle:" loading="lazy" width="20" height="20">去除系统偏好设置的小红点</h2>
<p>如果「系统设置」图标上仍有提示红点，可以执行以下命令去除：</p>
<pre><code class="lang-auto">defaults write com.apple.systempreferences AttentionPrefBundleIDs 0
killall Dock
</code></pre>
<p>执行后红点会消失。</p>
