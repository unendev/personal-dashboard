# Tactical Board 布局重设计

## 目标
将 TacticalBoard 从上下两分的布局改为三分布局：
- **左上**：ToDo 列表
- **中间**：AI 主控（CommandCenter 风格）
- **右侧**：MD 编辑区（笔记）

## 当前布局
```
┌─────────────────────────────────┐
│      TACTICAL BOARD             │
├─────────────────────────────────┤
│                                 │
│         ToDo 列表               │
│                                 │
├─────────────────────────────────┤
│                                 │
│      MD 编辑区（笔记）          │
│                                 │
└─────────────────────────────────┘
```

## 新布局
```
┌──────────────┬──────────────┬──────────────┐
│              │              │              │
│   ToDo       │   AI 主控    │   MD 编辑    │
│   列表       │  (Command    │   区（笔记） │
│              │   Center)    │              │
│              │              │              │
└──────────────┴──────────────┴──────────────┘
```

## 实现步骤

### 1. 布局结构
- 使用 `flex` 三分布局
- 左侧：`w-1/4` (25%)
- 中间：`w-1/2` (50%)
- 右侧：`w-1/4` (25%)

### 2. 左上 ToDo 区域
- 保留现有的 ToDo 功能
- 添加滚动条
- 保持现有的样式

### 3. 中间 AI 主控
- 参考 `CommandCenter.tsx` 的实现
- 显示 AI 对话/命令
- 支持实时交互

### 4. 右侧 MD 编辑区
- 保留现有的笔记编辑功能
- 添加大纲面板（左侧）
- 支持 Markdown 预览

## 关键组件

### MarkdownOutline
- 自动生成 Markdown 大纲
- 支持点击跳转
- 支持展开/收起

### CommandCenter 集成
- 复用 AI 主控逻辑
- 保持赛博朋克风格
- 支持工具调用

## 样式参考
- 使用 GOC 风格（赛博朋克）
- 青色主题（cyan）
- 深色背景（zinc-950）
- 玻璃态效果

## 优先级
1. 布局结构（高）
2. 左上 ToDo 功能（高）
3. 右侧 MD 编辑 + 大纲（高）
4. 中间 AI 主控（中）
