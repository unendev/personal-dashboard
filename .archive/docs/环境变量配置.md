# 环境变量配置说明

## 演示模式配置

### NEXT_PUBLIC_DEMO_MODE

控制是否启用演示模式。

- **默认值**: `true`（启用）
- **作用**: 启用后，未登录的用户将自动使用示例账户访问系统，无需手动登录
- **禁用方法**: 在 `.env.local` 文件中添加：

```env
NEXT_PUBLIC_DEMO_MODE=false
```

## 演示模式说明

### 启用演示模式（默认）

当 `NEXT_PUBLIC_DEMO_MODE` 为 `true` 或未设置时：

1. ✅ HR或访客无需登录即可查看完整功能
2. ✅ 自动使用示例账户（demo@example.com）
3. ✅ 包含预设的演示数据（待办、计时、宝藏等）
4. ✅ 降低访问摩擦，方便演示和展示

### 禁用演示模式

当 `NEXT_PUBLIC_DEMO_MODE` 设置为 `false` 时：

1. 🔒 必须登录才能访问系统功能
2. 🔒 未登录用户会被重定向到登录页面
3. 🔒 适合生产环境和私人使用
