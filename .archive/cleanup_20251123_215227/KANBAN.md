# 海龟汤游戏开发看板

## Backlog (产品待办列表)

### 游戏化体验增强
- 故事主题分类与精选 (例如：怪谈、悬疑、日常)。
- 在故事选择页面优先展示特定主题 (例如：将怪谈类故事放到最前面)。
- 设计更具沉浸感的游戏界面和交互动画。
- 增加音效或背景音乐，提升游戏氛围。
- 增加用户头像/昵称自定义功能。

### 核心功能迭代
- 用户可提交自己的故事 (UGC)，并经过审核后发布。
- 实现多人游戏模式，支持房间创建、邀请和匹配。
- 引入游戏内成就系统或排行榜。
- 支持故事保存和进度加载。

### 技术优化
- 优化 DeepSeek LLM 的 Prompt Engineering，提高主持人规则遵守度。
- 引入 DeepSeek LLM 的微调，以适应游戏特定语境。
- 搭建独立的微服务用于故事管理，解耦前后端。
